## Promo Tile Instructions

For the **440x280 Small Promo Tile**:
1. Open `promo-tile-440x280.html` in Chrome
2. Set browser zoom to 100%
3. Screenshot the page at exactly 440x280
4. Save as JPEG (no alpha/transparency)

The 1400x560 Marquee tile is optional — skip it for initial submission.
